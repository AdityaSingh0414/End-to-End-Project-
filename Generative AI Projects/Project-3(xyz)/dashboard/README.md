# Power BI Dashboard Plan

Use `powerbi_model_inputs.csv` as the first Power BI source.

Recommended pages:

1. Executive summary: total patients, high-risk cases, top disease, model accuracy KPI.
2. Disease distribution: disease counts by age group, gender, and month.
3. Prediction analytics: confidence histogram and drill-down by risk level.
4. Patient trends: visit timeline and abnormal vitals.

The `powerbi.pbix` file is intentionally a placeholder because PBIX is a binary Power BI Desktop artifact. Build it from the CSV export and keep the final binary here.
