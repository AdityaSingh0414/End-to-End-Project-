# AnalystGPT

An AI-Powered Conversational Business Intelligence Platform.

## Features
- **Chat With Data**: Natural language to SQL queries.
- **Dashboard**: Interactive data visualizations.
- **Automated EDA**: Automated Exploratory Data Analysis.
- **Data Quality Analyzer**: Check for nulls, duplicates, and outliers.
- **Forecasting**: ML-based forecasting for revenue/demand.
- **Customer Segmentation**: KMeans-based customer clustering.
- **RAG Assistant**: Document Q&A using FAISS and Sentence Transformers.
- **Report Generator**: Export insights to PDF, Excel, and CSV.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Groq API key to `.env` file.
4. Run the app: `streamlit run frontend/app.py`
